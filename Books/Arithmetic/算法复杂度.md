- 算法：有限的操作序列。

- 算法的五大特性：有穷性、确定性、可行性、输入、输出。

- 好算法的目标要求：正确性、可读性、健壮性、高效率、低存储量。

| 算法复杂度      | 说明                   |
| --------------- | ---------------------- |
| 时间复杂度 T(n) | 执行语句的总步数       |
| 空间复杂度 O(n) | 算法所需存储空间的量度 |
