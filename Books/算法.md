# 算法基础

- 算法：有限的操作序列。

- 算法的五大特性：有穷性、确定性、可行性、输入、输出。

- 好算法的目标要求：正确性、可读性、健壮性、高效率、低存储量。

| 算法复杂度           | 说明                   |
| -------------------- | ---------------------- |
| 时间复杂度<br />T(n) | 执行语句的总步数       |
| 空间复杂度<br />O(n) | 算法所需存储空间的量度 |

# 分治策略

| 分治策略步骤    | 说明                                         |
| --------------- | -------------------------------------------- |
| 分（Divide）    | 将问题划分为若干规模较小、形式相同的子问题。 |
| 治（Conquer）   | 递归求解，若问题规模够小，则直接求解。       |
| 合并（Combine） | 子问题的解构建原问题的解。                   |

> 最大子数组。
