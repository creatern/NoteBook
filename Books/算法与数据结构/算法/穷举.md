# 鸡兔同笼

- 今有鸡兔同笼，上有三十五头，下有九十四足，问鸡兔几何？

```java
package exhaustive;

public class JiTu {
    public static void main(String[] args) {
        count(35,94);
    }

    public static void count(int head, int foot) {
        for (int i = 0; i < head; i++) { //i 为鸡头数量
            if ((i * 2 + (head - i) * 4) == foot) {
                System.out.println("鸡：" + i + "兔：" + (head - i));
            }
        }
    }
}
```
