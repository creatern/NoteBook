# PLSQl Developer连接远程数据库、

## 1. 安装oracle Clinet 

1. instantclient-basic-nt-11.2.0.4.0.zip
2. 解压到PLSQL的安装位置：D:\PLSQL Developer

## 2. 配置tnsnames.ora

-  在D:\PLSQL Developer\instantclient_11_2创建network\admin这两个文件夹，然后再创建tnsnames.ora文件
- 编辑tnsnames.ora，内容如下

```
ORCL=  
  (DESCRIPTION =  
    (ADDRESS = (PROTOCOL = TCP)(HOST = 192.168.149.128)(PORT = 1521))  
    (CONNECT_DATA =  
      (SERVER = DEDICATED)  
      (SERVICE_NAME = orcl)  
    )  
  )
  

ORCL：连接名，连接数据库的别名。
ADDRESS：服务器的地址  
PROTOCOL：使用的协议，这里是TCP的  
HOST: 就是远程数据库的地址
PORT：服务器的监听端口号  
SERVICE_NAME：服务器服务名称SID
```

## 3. 修改PLSQL所在的主机环境变量

**添加环境变量：**

- NLS_LANG = SIMPLIFIED CHINESE_CHINA.ZHS16GBK 或 AMERICAN_AMERICA.ZHS16GBK
- TNS_ADMIN  = D:\PLSQL Developer\instantclient_11_2\network\admin

**在Path后增加：**

- D:\PLSQL Developer\instantclient_11_2

## 4. 修改PLSQL配置

**Tools -> perference -> Connection**

- Oracle Home：D:\Oracle\instantclient_11_2
- OCI Library：D:\Oracle\instantclient_11_2\oci.dl
